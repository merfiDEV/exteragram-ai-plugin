# Gemini AI Plugin для exteraGram

AI-плагин для exteraGram с командами для Gemini и OpenRouter.

## Команды

- `.g <запрос>`: отправить запрос в Google Gemini
- `.o <запрос>`: отправить запрос в OpenRouter
- `.gfix <текст>`: исправить пунктуацию и орфографию через Gemini
- `.ofix <текст>`: исправить пунктуацию и орфографию через OpenRouter

## Возможности

- Поддержка провайдеров Gemini и OpenRouter
- Единый router команд вместо дублирующихся обработчиков
- Общий pipeline доставки ответа
- Более понятные ошибки API и сети
- Блокировка параллельных запросов для защиты от race conditions
- Режим вставки ответа в поле ввода вместо немедленной отправки
- Конвертация Markdown для обычных ответов
- Plain text-режим для команд исправления текста

## Настройки

- `Gemini API key`
- `OpenRouter API key`
- `OpenRouter model`
- `Сразу отправлять ответ`
- `Позиция уведомления`

## Установка

1. Сохраните `gemini_ai.plugin`.
2. Отправьте файл в любой чат в exteraGram.
3. Нажмите на файл и выберите установку.
4. Откройте `exteraGram Preferences -> Plugins` и включите плагин.

## Примечания

- Модель Gemini по умолчанию: `gemini-2.5-flash-lite`
- Модель OpenRouter по умолчанию: `openai/gpt-5.2`
- Плагин пока сохраняет текущий кастомный popup UI для уведомлений
